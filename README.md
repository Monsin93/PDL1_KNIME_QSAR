# PD-L1 Machine Learning Workflow (Supporting Repository)

This repository provides the supporting materials associated with the manuscript:

“Open-Source Machine-Learning Framework for Predicting PD-L1 Inhibitory Potency Using Curated Bioactivity Data and Molecular Fingerprints.”

The purpose of this repository is to support transparency and methodological clarity by sharing curated datasets and workflow illustrations used in the study.
All detailed procedures are fully described in the manuscript and Supporting Information.

Contents of This Repository

The repository contains the following files:

## Supporting Information

Supporting Information.docx

Detailed description of data curation, feature selection, model development, and validation workflows

KNIME workflow diagrams illustrating each step of the machine-learning pipeline

## Curated Datasets

ChEMBL_curated_dataset (csv file)

Curated PD-L1 bioactivity dataset derived from the ChEMBL database

PDL1_Data_trainingset (csv file)

PDL1_Data_testset (csv file)

PDL1_Data_externalvalidationset (csv file)

External validation dataset used to assess model generalization
